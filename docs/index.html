How alerts reach you

Notifications are sent through ntfy — open-source, no account or password. Install the ntfy app, subscribe to your secret topic, and set that topic as the NTFY_TOPIC GitHub secret. Alerts then pop up on your phone like a text.

Save your watchlist

Credential-free save:
  1. Click Copy config.json.
  2. Click Open config.json on GitHub (you're already logged in there).
  3. Select-all in the editor, paste, and hit Commit changes. The next run uses it. ✅
